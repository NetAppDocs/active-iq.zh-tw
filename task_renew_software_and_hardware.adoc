= 更新儲存系統的軟體和硬體
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
您可以主動識別已過期或即將在未來6個月內到期的軟體和硬體、並傳送更新硬體和軟體的要求。

.步驟
. 按一下「*規劃*」小工具中的「*續約*」。
. 選取您要續約的系統、然後按一下*更新*。
. 您也可以提供其他意見。
. 按一下*傳送*。
