= 在 Digital Advisor 中確定是否建議對您的儲存系統進行系統重新整理
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
如果您想確定是否建議您對 Tech Support 合約或硬體進行系統更新，可以使用系統更新選項。

您可以透過生命週期規劃（*儲存* > *生命週期規劃*）或 Digital Advisor（*儀表板* > *規劃小工具* > *系統重新整理*）存取此功能。

如需此功能的詳細資訊，請參閱主控台文件中的 link:https://docs.netapp.com/us-en/bluexp-economic-efficiency/use/tech-refresh.html["評估系統重新整理"]。
