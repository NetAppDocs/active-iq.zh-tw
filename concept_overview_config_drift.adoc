[role="lead"]
Config DrivFT功能可將系統範本與「golden」或基礎系統範本進行比較、以識別組態差異。您可以排程每週或每月的飄移報告、或是隨需產生報告。您可以使用組態飄移報告中提供的「可執行教戰手冊」來減輕一些差異。

此功能僅適用於持有顧問與專家支援合約的系統。

您可以查看下圖、瞭解識別組態差異並產生報告的工作流程。
